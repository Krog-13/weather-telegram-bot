## Weather TeleBot
This bot shows the weather in the city of Yaroslavl.

Weather data: 
- Average temperature
- Probability of precipitation
- Average pressure
- Average relative humidity


### Instruction
To connect the bot, enter the telegram app 
username *@Gashanbot*\
This bot shows 16 days Weather Forecast
